# 🔌 AI-Based Smart Residential Electricity Monitoring System

## 🚀 Overview
This project is an AI-powered smart electricity monitoring system that uses ESP32, Blynk IoT, and Python to monitor, analyze, and control energy consumption in real time.

---

## 🧠 Features
- Real-time voltage, current, power monitoring  
- AI-based energy prediction  
- Sudden spike detection  
- Automatic relay control  
- Smart billing estimation  

---

## 🛠️ Technologies Used
- ESP32  
- PZEM Energy Sensor  
- Python  
- Blynk IoT Platform  

---

## 📁 Project Structure
- ESP32_Code → Arduino code for data collection  
- Python_AI → AI logic and control system  
- Images → Setup and dashboard visuals  

---

## ⚙️ Working
ESP32 → Blynk Cloud → Python AI → Control Signal → Relay
